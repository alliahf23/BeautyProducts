# Glowify Beauty Shop

A simple static website for a beauty and skincare shop, inspired by your provided layout but with a fresh lavender and rose color palette.

## How to view

- Open the `index.html` file directly in your browser (Edge, Chrome, etc.).
- Make sure `styles.css` is in the same folder so the page has styling.

## Features

- Shop header with navigation and promo bar
- Category filter pills for **Face Care**, **Serums**, **Body Care**, **Masks**, and **Hair Care**
- Product cards with images, prices, descriptions, and **Add to cart** buttons
